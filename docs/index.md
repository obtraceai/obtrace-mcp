# Obtrace MCP Server Documentation

## Overview

The Obtrace MCP Server exposes the Obtrace observability platform as a set of MCP (Model Context Protocol) tools. This allows LLM-powered agents, IDE extensions, and AI assistants to interact with your observability data programmatically.
